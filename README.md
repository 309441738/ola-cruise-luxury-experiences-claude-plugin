# Luxury Cruise Experiences · Claude Code 插件

Indulge in premium luxury cruise experiences that offer top-notch amenities and exclusive services. Tailored for discerning travelers seeking the finest.

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/ola-cruise-luxury-experiences-claude-plugin
/plugin install ola-cruise-luxury-experiences@ola-cruise-luxury-experiences-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **Luxury Cruise Experiences**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
